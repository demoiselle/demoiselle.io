Versão Candidata 1.2.0-RC1 Liberada!

Novas Funcionalidades

Novo Mecanismo de Descoberta da Cadeia de Autoridades do Certificado

Em várias situações o componente precisa ter certeza que o certificado possui a cadeia de certificados das Autoridades Certificadoras corretas, e para isso é necessário que o componente saiba quais existem.

Anteriormente somente os certificados gerados pelas Autoridades Certificadoras que estivessem dentro do componente (por isso ele precisava ser atualizado com maior frequência) é que eram validados, agora o componente também tenta encontrar a cadeia de outras maneiras, seguindo a seguinte ordem de descoberta:

  • Dentro do componente
  • Home do usuário
  • Busca no Serpro
  • Busca no ITI
Somente quando em nenhum destes locais possuir todos os certificados para montar a cadeia de autoridades certificadoras do certificado do usuário é que ocorre algum erro.
Atualização para Java 7

Agora a versão necessária para rodar o componente é a versão 7 do Java.

Correção da validação da identificação da política (Hash e Links da Política)

Ao validar alguns tipos de assinatura com a versão anterior ocorria um erro.

Como Atualizar

Para testar a nova versão do componente certificate em seu projeto atualize suas dependencias do maven confme as sugestões abaixo: 

<dependency>
	<groupId>br.gov.frameworkdemoiselle.component</groupId>
	<artifactId>demoiselle-certificate-core</artifactId>
	<version>1.2.0-RC1</version>
</dependency>
<dependency>
	<groupId>br.gov.frameworkdemoiselle.component</groupId>
	<artifactId>demoiselle-certificate-ca-icpbrasil</artifactId>
	<version>1.2.0-RC1</version>
</dependency>
<dependency>
	<groupId>br.gov.frameworkdemoiselle.component</groupId>
	<artifactId>demoiselle-certificate-signer</artifactId>
	<version>1.2.0-RC1</version>
</dependency>

Para outras partes do componente como Applet e Desktop também é possível atualizar para a versão 1.2.0-RC1.

Após adicionar é importante atualizar as configurações do Maven clicando sobre o projeto com o botão direito do mouse, acessando as opções do Maven e por fim clicando em Update Project. Na tela que será aberta selecione o seu projeto e clique em OK.

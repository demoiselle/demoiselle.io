1.1. Uso do componente

Para utilizar o componente Signer-Core num projeto Java, basta adicionar a sua dependência no arquivo pom.xml , conforme o seu gerenciador de projetos:

  • Apache-Maven

            		<dependency>
				<groupId>org.demoiselle.signer</groupId>
				<artifactId>core</artifactId>
				<version>3.2.4</version>
			</dependency>

  • Apache Buildr

    'org.demoiselle.signer:core:jar:3.2.4'

  • Apache Ivy

    <dependency org="org.demoiselle.signer" name="core" rev="3.2.4" />

  • Groovy Grape

    @Grapes(@Grab(group='org.demoiselle.signer', module='core', version='3.2.4'))

  • Gradle/Grails

    <dependency org="org.demoiselle.signer" name="core" rev="3.2.4" />

  • Scala SBT

    libraryDependencies += "org.demoiselle.signer" % "core" % "3.2.4"

  • Leiningen

    [org.demoiselle.signer/core "3.2.4"]

Caso não esteja utilizando nenhum outro tipo de gerenciador (estava morando numa caverna nos últimos dez anos), pode baixar o .jar do repositório:

http://repo1.maven.org/maven2/org/demoiselle/signer/core/

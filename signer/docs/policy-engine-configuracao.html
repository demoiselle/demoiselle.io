7.1. Instalação do componente

Para utilizar o componente policy-engine num projeto Java, basta adicionar a sua dependência no arquivo pom.xml , conforme o seu gerenciador de projetos:

  • Apache-Maven

            		<dependency>
				<groupId>org.demoiselle.signer</groupId>
				<artifactId>policy-engine</artifactId>
				<version>3.1.3</version>
			</dependency>

  • Apache Buildr

    'org.demoiselle.signer:policy-engine:jar:3.1.3'

  • Apache Ivy

    <dependency org="org.demoiselle.signer" name="policy-engine" rev="3.1.3" />

  • Groovy Grape

    @Grapes(@Grab(group='org.demoiselle.signer', module='policy-engine', version='3.1.3'))

  • Gradle/Grails

    <dependency org="org.demoiselle.signer" name="policy-engine" rev="3.1.3" />

  • Scala SBT

    libraryDependencies += "org.demoiselle.signer" % "policy-engine" % "3.1.3"

  • Leiningen

    <[org.demoiselle.signer/policy-engine "3.1.3"]

Caso não esteja utilizando nenhum outro tipo de gerenciador (estava morando numa caverna nos últimos dez anos), pode baixar o .jar do repositório:

http://repo1.maven.org/maven2/org/demoiselle/signer/policy-engine/
